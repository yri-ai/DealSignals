# SEC Filing Research Guide - WeWork/BowX Merger

## What to Look for in Each Filing Type

### BowX Acquisition Corp Filings

#### S-4 (Registration Statement)
**This is THE document for merger analysis**

**Where to find the gold:**
1. **Projections** (usually in "Summary" or "Certain Projected Financial Information")
   - Revenue forecasts 2021-2025
   - EBITDA projections
   - Cash flow assumptions
   - Key operational metrics (members, locations, occupancy)

2. **Risk Factors** (dedicated section)
   - Lease liability risks
   - Business model risks
   - Market risks
   - Integration risks
   - Check what they DIDN'T emphasize vs. what killed them

3. **Deal Terms**
   - Valuation ($9B enterprise value)
   - Earnout provisions
   - PIPE investment details
   - Sponsor compensation
   - Redemption provisions

4. **Management Discussion**
   - How they justified the valuation
   - Key assumptions underlying projections
   - Comparison to competitors

5. **Background of the Merger**
   - Due diligence timeline
   - What was investigated
   - Alternative transactions considered

**Search tips:**
- Ctrl+F: "projection", "forecast", "estimate"
- Ctrl+F: "risk factor"
- Ctrl+F: "lease", "liability", "obligation"
- Look for tables with 5-year forecasts

---

#### DEFM14A (Definitive Proxy Statement)
**Final shareholder vote materials - may contain updates to S-4**

**What to check:**
1. **Fairness Opinion**
   - Investment bank's valuation analysis
   - Methodologies used (DCF, comparable companies, precedent transactions)
   - Key assumptions

2. **Updated Financials**
   - Any changes from S-4 filing
   - Recent quarterly results
   - Material changes since initial filing

3. **Voting Requirements**
   - Shareholder approval thresholds
   - Sponsor voting agreements

4. **Interests of Directors/Officers**
   - Conflicts of interest
   - Compensation tied to deal

**Search tips:**
- Ctrl+F: "fairness opinion"
- Ctrl+F: "valuation"
- Compare projections to S-4 version

---

#### 8-K (Current Reports)
**Material events - multiple filings**

**Key 8-Ks to find:**

1. **Merger Announcement** (~March 2021)
   - Initial deal terms
   - Press release
   - Investor presentation (Exhibit 99.1)

2. **Merger Closing** (~October 2021)
   - Final terms
   - Actual redemptions
   - PIPE investment closing

3. **Earnings Releases** (2021-2023)
   - Quarterly results vs. projections
   - Management commentary
   - Updated guidance

**What to track:**
- Date of each material event
- How story changed over time
- When guidance was reduced/eliminated
- Tone of management commentary evolution

### WeWork Inc Filings (Post-Merger)

#### 10-K (Annual Reports)
**Critical for tracking actual vs. projected performance**

**Priority sections:**

1. **Item 7: Management's Discussion & Analysis (MD&A)**
   - Results of operations
   - Liquidity and capital resources
   - Critical accounting estimates
   - Compare to SPAC projections

2. **Financial Statements**
   - Balance sheet (focus on lease liabilities)
   - Income statement (revenue vs. projections)
   - Cash flow statement (cash burn rate)
   - Notes to financial statements

3. **Item 1A: Risk Factors**
   - Compare to SPAC-era risk disclosures
   - New risks that emerged
   - How risk language evolved

4. **Item 7A: Market Risk**
   - Interest rate exposure
   - Foreign currency exposure

**Years to collect:**
- 10-K for 2021 (partial year post-merger)
- 10-K for 2022 (full year)
- 10-K for 2023 (if filed before bankruptcy)

**Analysis framework:**
```
| Metric           | SPAC Projection | 2021 Actual | 2022 Actual | 2023 Actual | Variance |
|------------------|-----------------|-------------|-------------|-------------|----------|
| Revenue          |                 |             |             |             |          |

| Members          |                 |             |             |             |          |
| Locations        |                 |             |             |             |          |
| Cash Position    |                 |             |             |             |          |
| Lease Liability  |                 |             |             |             |          |
```

#### 10-Q (Quarterly Reports)
**Track deterioration in real-time**

**Priority quarters:**
- Q4 2021 (first full quarter post-merger)
- All quarters 2022
- Q1-Q3 2023 (before bankruptcy)

**What to extract:**
- Sequential revenue trends
- Membership trends
- Cash burn rate
- Lease liability changes
- Any going concern warnings
- MD&A discussion of headwinds

**Warning signs to track:**
- When did covenant violations start?
- When did liquidity concerns emerge?
- When did they stop giving guidance?
- Changes in accounting estimates

---

#### 8-K (Current Reports)
**Real-time material events**

**Types to prioritize:**

1. **Earnings announcements**
   - Results vs. expectations
   - Guidance changes
   - Management commentary

2. **Leadership changes**
   - CEO/CFO departures
   - Board changes

3. **Financing events**
   - Debt raises
   - Covenant waivers
   - Asset sales

4. **Material agreements**
   - Lease restructurings
   - Strategic partnerships
   - Bankruptcy warning signs

## Research Questions by Filing

### From S-4/DEFM14A:
- [ ] What was projected revenue for 2022? 2023? 2024?
- [ ] What was projected EBITDA?
- [ ] What were key operational assumptions (member growth, pricing, occupancy)?
- [ ] How were lease liabilities presented/explained?
- [ ] What risk factors were disclosed about leases?
- [ ] What was the valuation methodology?
- [ ] What comparables were used?

### From 10-Ks/10-Qs:
- [ ] What was actual revenue by year/quarter?
- [ ] What was actual EBITDA/net income?
- [ ] How did membership trend?
- [ ] How did lease liabilities grow/change?
- [ ] When did going concern language appear?
- [ ] What explanations were given for misses?
- [ ] How did cash position deteriorate?

### From 8-Ks:
- [ ] When was guidance withdrawn?
- [ ] What were quarterly misses vs. expectations?
- [ ] When did leadership turnover occur?
- [ ] What financing was raised (desperation capital)?
- [ ] What were warning signs in MD&A?

## Tips for Reading SEC Filings

1. **Use Adobe Acrobat or browser search liberally**
   - Search terms: projection, forecast, estimate, guidance
   - Search: risk, material, adverse
   - Search: lease, liability, obligation

2. **Focus on tables and charts**
   - Financial statement notes
   - Projection tables in S-4
   - Reconciliation tables in 10-Ks

3. **Read MD&A carefully**
   - This is where management tells the story
   - Watch how narrative changes over time

4. **Don't skip footnotes**
   - Lease accounting details are in Note X
   - Commitments and contingencies

5. **Compare filings side-by-side**
   - S-4 projections vs. 10-K actuals
   - Risk factors evolution
   - Management tone changes

## Common Locations in Filings

**In S-4:**
- Projections: Usually page 50-100, in "Summary" section
- Risk Factors: Usually page 40-80, dedicated section
- Deal Terms: "The Business Combination Agreement" section

**In 10-K:**
- MD&A: Item 7 (usually page 30-60)
- Financial Statements: Item 8 (usually starts page 60-80)
- Risk Factors: Item 1A (usually page 10-30)
- Lease disclosures: Notes to financial statements (Note 8 or 9 typically)

**In 10-Q:**
- MD&A: Part I, Item 2
- Financial Statements: Part I, Item 1
- Risk Factors: Part II, Item 1A (only if updated)
